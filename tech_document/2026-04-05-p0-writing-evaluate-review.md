## Part 2: 원고 리뷰 (/review)

### 총평

도입 챕터로서 매우 성공적인 종합이다. ai-writing 프로젝트의 5개 슬라이드쇼
(총 1,000+ 슬라이드 분량)를 302줄의 응집력 있는 단일 챕터로 압축했다.
36,000년 기록 역사에서 시작해 AI 시대 재현가능 문서 파이프라인까지 도달하는
서사적 아크(narrative arc)가 명확하다.

특히 세 가지 핵심 프레임워크가 효과적으로 작동한다:
1. **3차 비용 붕괴**: 복제→편집→생성 비용의 순차적 붕괴
2. **3단계 수렴**: WYSIWYG → 도메인 마크업 → 텍스트+AI
3. **역할 진화**: Writer → Editor → Architect → Orchestrator

### 섹션별 리뷰

#### §기록은 왜 시작되었는가 (11-29줄) — 우수

동굴벽화에서 알파벳까지의 역사를 "기억의 외부화"라는 단일 동기로 묶은 것이
효과적이다. "매체 + 문자 = 문서 시스템" 공식은 이후 전체 챕터의 분석 틀이 된다.

**개선 제안:** 반구대 암각화(BC 5,000) 등 한국 사례를 추가하면 독자 공감대 강화.

#### §종이와 인쇄 혁명 (33-47줄) — 양호

직지심체요절→구텐베르크 서사가 간결하고 정확하다. "매체의 전파 속도 = 문명의
전파 속도"라는 통찰이 좋다.

**개선 제안:** 인쇄 혁명의 파급 효과(종교개혁, 과학혁명 촉발)를 1-2문장 추가.

#### §타자기에서 디지털 텍스트로 (49-63줄) — 보통

13줄로 다른 섹션 대비 짧다. QWERTY, 80칼럼, 문자 인코딩을 다루나 속도감이 빠름.

**개선 제안:**
- 한글 입력 역사(두벌식/세벌식 논쟁)를 2-3줄 추가
- HWP(1989)의 한국적 맥락을 언급 (§WYSIWYG에서 중복 가능성 확인 후)

#### §디지털 화석 (65-92줄) — 우수

경로 의존성의 학술적 토대(David 1985, Arthur 1994)와 생생한 사례(QWERTY, hjkl,
CR+LF, 플로피, 기호)의 결합이 탁월하다. "오늘 만드는 선택이 100년 뒤의 화석이
된다"는 인용구가 챕터 전체를 관통하는 메시지로 기능한다.

**개선 제안:** 특별히 없음. 가장 완성도 높은 섹션.

#### §문서 엔지니어링 (94-156줄) — 우수

Glushko C·A·C 모델의 설명이 명확하고, Quarto shortcode/include/params로의
대응이 직관적이다. 서사/하이브리드/트랜잭션 3유형 진화가 동일한 3단계 패턴을
따른다는 통찰이 강력하다.

**개선 제안:**
- C·A·C → Quarto 대응을 보여주는 SVG 다이어그램 추가
- 8단계 방법론이 3줄 불릿 포인트로 요약되어 약간 축약적. 분석-설계-구현 각각
  1문장씩 예시 추가 가능

#### §WYSIWYG에서 WYAIWYM으로 (158-201줄) — 우수

3단계 패러다임 전환(WYSIWYG→WYSIWYM→WYAIWYM)이 깔끔하게 정리되었다.
특히 "커누스의 WEB = AI 시대의 프롬프트"라는 연결이 독창적이다.
Docs as Code → X as Code 확산도 잘 다루었다.

**개선 제안:**
- WYAIWYM은 자체 조어(coined term)인데, 출처/정의를 좀 더 명확히 하면 좋겠음
- Scribe(1980, Brian Reid)에 대한 참고문헌 부재 (bib 항목 추가 권장)

#### §AI 글쓰기의 5단계 스펙트럼 (203-228줄) — 우수

Lv.0~Lv.4 스펙트럼 표가 직관적이다. "10-20일 → 3-5시간, 50-100배 가속"이라는
수치가 설득력 있다. Writer→Editor→Architect→Orchestrator 역할 진화 서사가
패러다임 전환 섹션과 일관성을 유지한다.

**개선 제안:** 5단계 스펙트럼이 ai-writing 프로젝트의 독자 분류인지, 학술적
출처가 있는지 명시하면 신뢰도 향상.

#### §인간 중심 공동저작의 원칙 (230-256줄) — 양호

ACM 2025 연구와 Stanford HAS의 핵심 발견을 정확히 전달한다. "47.5%가 기술적으로
필요한 것보다 더 높은 인간 주도권을 선호"라는 데이터 포인트가 강력하다.

**개선 제안:**
- Flower & Hayes(1981) 인지적 글쓰기 이론[@FlowerHayes1981]이 bib에 추가되었으나
  본문에서 인용되지 않음. 4대 설계 전략의 이론적 기반으로 언급하면 학술적 깊이 향상
- HAS 5단계를 시각화하는 SVG 추가 가능

#### §재현가능 문서 파이프라인 (258-302줄) — 양호

재현성 위기 통계(Baker 2016, OSC 2015, Pimentel 2019)가 효과적으로 배치되었다.
"88.4% 에러이지만 진보"라는 역설적 해석이 통찰력 있다.
5단계 파이프라인과 4대 기둥 구조가 명확하다.

**개선 제안:**
- 세 지평선(H1/H2/H3)이 확실성 수치와 함께 제시되어 좋으나, 근거가 명시되지
  않음 (ai-writing 프로젝트의 자체 분류인지 외부 프레임워크인지)
- 마무리 문장("도구는 바뀌지만 글쓰기의 본질은 변하지 않는다")이 좋으나,
  후속 챕터(p1-intro)로의 전환을 안내하는 1문장 추가 가능
