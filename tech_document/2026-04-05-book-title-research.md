## 2. 시리즈 진화 구조

```
AI 글쓰기                    →  AI 에이전트 글쓰기
─────────────────────           ─────────────────────
도구(Quarto/Typst)가             에이전트(Claude Code/Cursor)가
사고를 바꾸는                    코딩을 바꾸는
과학기술 저작                    소프트웨어 개발

글쓰기 = 문서 작성               글쓰기 = 코드 작성
도구 중심                        에이전트 중심
수동 파이프라인                   자율 워크플로우
```

**"글쓰기" 확장 해석**: 전작에서 "글쓰기 = 문서 저작"이었다면, 후속작에서는 "글쓰기 = 코드 저작"으로 자연스럽게 확장된다. 프로그래밍은 결국 코드를 **쓰는** 행위이며, 에이전트 시대에 "누가 쓰는가"가 바뀌는 것이 핵심 메시지다.
